# Email Spam Detection System using NLP

## Problem Statement
Spam emails cause security and productivity issues. This project builds an automated system to classify emails as spam or legitimate using NLP and machine learning.

## Tasks Covered
- Collect email datasets
- Preprocess email text
- Apply NLP techniques
- Train spam classifier
- Evaluate accuracy

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python src/main.py

## Dataset
SMS Spam Collection Dataset (Kaggle/UCI)

## Results
Accuracy ~97%
